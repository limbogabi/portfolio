QA Run Tracker

A full-stack QA management platform for creating and tracking test cases and test runs. Built with a modern MERN-style stack: React + Vite on the frontend and Node.js + Express with MongoDB Atlas on the backend. The backend API is hosted on Render for full cloud availability.

  • Frontend deployed via GitHub Pages
  • Backend hosted on Render (Node.js/Express)
  • Database: MongoDB Atlas
  • Create, delete, and manage test cases
  • Create test runs dynamically from UI
